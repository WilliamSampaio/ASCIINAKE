# ASCIINAKE

## Build

```bash
g++ -Wall -Werror src/*.cpp -o asciinake -lncursesw -ltinfo
```

Or

```bash
chmod +x build.bash && ./build.bash
```

## Play/Install

```bash
chmod +x install.bash && sudo ./install.bash
source ~/.zshrc # or .bashrc
asciinake # to play!
```
